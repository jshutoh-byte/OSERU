# PDF Stamp Utility (Lightweight Edition)

業務上のPDFや画像ファイルに対して、電子印影やサイン（スタンプ）を直感的に合成するためのデスクトップツールです。

## ■ 概要
「特定の1箇所に、素早く正確にスタンプを置きたい」という実務ニーズに特化し、既存の重厚なPDF編集ソフトを起動する手間を省くために開発されました。
OpenCVによる直感的なドラッグ＆ドロップ操作と、PyMuPDFによる高精度なPDF出力を組み合わせています。

## ■ 主な機能と特徴
* **マルチフォーマット対応**: PDFだけでなく、一般的な画像形式（PNG, JPG）への合成も可能です。
* **直感的なUI**: マウス操作でスタンプの位置・サイズを自由に変更でき、プレビューを見ながら確定できます。
* **日本語パス対応**: 日本のビジネス現場で発生しがちな「日本語フォルダ・ファイル名」によるエラーを独自実装で回避しています。
* **最前面ダイアログ**: 操作ミスを防ぐため、確認メッセージが他のウィンドウに隠れない設計になっています。

## ■ 使い方
1. プログラムを実行します。
2. ベースとなるファイル（PDF/画像）と、合成したいスタンプ画像を選択します。
3. 編集画面が開いたら、以下の操作を行います：
   - **マウスドラッグ**: スタンプの位置移動
   - **右下ハンドルドラッグ**: スタンプのサイズ調整
   - **Enterキー**: 位置の確定・保存
   - **Qキー**: 保存せずに終了
4. 生成されたファイルは `output` フォルダへ自動保存されます。

## ■ 技術仕様
* **Language**: Python 3.x
* **Libraries**: 
  - `PyMuPDF (fitz)`: PDF構造の解析と高精細な画像挿入
  - `OpenCV (cv2)`: 高速な画像処理とリアルタイムプレビューUI
  - `Tkinter`: ネイティブなファイル選択ダイアログの提供

## ■ 免責事項
* 本ツールは実務の効率化を目的としていますが、出力結果は必ず利用者が最終確認を行ってください。
* PDFのセキュリティ設定によっては編集できない場合があります。
