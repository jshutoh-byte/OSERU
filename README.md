# OSERU (旧: PDF Stamper)

「あれ、ハンコどこに押せばええんや…？」
そんな現場の困りごとを解決する、ズバッと！電子捺印支援ツールです。

## ■ 開発背景
「PDFにハンコを押す」という単純な作業でも、ソフトが英語だったり操作が難しかったりすると、現場では大きな負担になります。
「OSERU」は、ITに不慣れな方でも、まるでお餅に判子を押すような感覚で、直感的に電子印影を配置できるように開発されました。

## ■ 本ツールの特徴
* **完全日本語ガイド**: 画面内に日本語で操作方法が出るので、マニュアルを読み直す必要がありません。
* **マウスでポン！**: マウスドラッグで位置を合わせ、右下を引っ張るだけでサイズ調整が可能です。
* **PDFも画像もOK**: どちらの形式でも、まるで紙に押すように自然に保存されます。
* **除霊済み**: 現場を混乱させる英語メニューや複雑な設定を徹底的に排除しました。

## ■ 動作環境
- Windows 10/11
- Python 3.x
- 必要ライブラリ: `pymupdf`, `opencv-python`, `Pillow`

## ■ 使い方
1. **書類を選択**: 捺印したいPDFまたは画像を選びます。
2. **ハンコを選択**: 自分の印影画像（PNG推奨）を選びます。
3. **位置を調整**: 赤い枠をマウスで動かし、サイズを決めます。
4. **Enterで確定**: 「この位置でよろしいですか？」に「はい」と答えるだけで、outputフォルダに完成品が保存されます。
