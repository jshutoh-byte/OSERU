===============================================================
PDF Stamp Utility (Lightweight Edition)
===============================================================

作成者：J. Shuto (@jshutoh-byte)
バージョン：1.0
日付：2026-01-14

---------------------------------------------------------------
■ 概要
---------------------------------------------------------------
本ツールは、PDFや画像ファイルに対して、電子印影やサイン（スタンプ）を
直感的に合成するための実務特化型デスクトップツールです。

「特定の1箇所に、素早く正確にスタンプを置きたい」という現場のニーズに
応えるため、軽量かつ直感的な操作性を重視して開発されました。

---------------------------------------------------------------
■ 使い方
---------------------------------------------------------------
1. プログラム（exeまたはスクリプト）を実行します。
2. 画面の指示に従い、ベースとなるファイルとスタンプ画像を選択します。
3. プレビュー画面が表示されたら、以下の操作で調整を行います。
   - マウスドラッグ        ：スタンプの移動
   - 右下緑枠ドラッグ      ：スタンプのサイズ調整
   - Enterキー             ：位置を確定して保存
   - Qキー                 ：保存せずに終了
4. 生成されたファイルは「output」フォルダ内に保存されます。

---------------------------------------------------------------
■ 技術仕様
---------------------------------------------------------------
・PDF解析エンジン：PyMuPDF (fitz)
・画像処理・UI  ：OpenCV (cv2)
・ダイアログ    ：Tkinter

日本語を含むファイルパスにも対応しており、日本のビジネス現場で
発生しがちな読み込みエラーを独自ロジックにより回避しています。

---------------------------------------------------------------
■ 注意事項
---------------------------------------------------------------
・Windows環境専用のツールです。
・編集権限（パスワード等）がかかっているPDFには対応していません。
・本ツールは現状有姿(AS-IS)で提供されます。
　重要な書類を処理する際は、必ず事前にバックアップを推奨します。
